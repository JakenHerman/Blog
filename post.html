Possibly the easiest swap method for sorting in Java

I swear, it's really simple.

Posted by Jaken Herman on April 3, 2015

Alright, you've just learned your first sorting algorithms and you're ready to put them into practice, right? Sure. But there is one major thing holding you back from doing so. You don't know how to swap values in your array in order to carry out your sorting operations! Don't worry, I'll make this quick and painless so you can get back to work. The swap method I'm going to show you is something called the "slash, slash, backslash" method of swapping. Now, it's a pretty common swap, but it isn't typically called the "slash, slash, backslash" method. It is hands down the easiest way to remember how to swap two values in your array. Now, first, I'll write the code in Java, and then I'll explain why it's called slash, slash, backslash, then I'll explain why it works.

public void swapSSB(int[] a, int i, int j){
    int temp;
    temp = a[i];
    a[i] = a[j];
    a[j] = temp;
}
 

public void swapSSB(int[] a, int i, int j){
    int temp;
    temp = a[i];
    a[i] = a[j];
    a[j] = temp;
}


So why is this called the slash, slash, backslash method? It's pretty simple, really. If you draw an imaginary slash (/) from a[i] to a[i] as well as one directly below it from a[j] to a[j], and then a backslash (\) from temp to temp, it really helpts to remember the structure of the swap. Why does it work?
It works because you're storing the value of a[i] in an integer "temp", then loading the value in a[j] to a[i]. You're then loading the previous value of a[i] back into a[j]. For example, lets say "i" was 0 and "j" was 1. Let's also pretend the array only holds 2 values: [3, 5]

Now, step by step let's walk through the swap. We declare an integer variable "temp", but we don't initialize it to anything yet. Then, we initialize it to the value of a[0] (because i is 0, remember?). So temp now equals 3, because a[0] = 3. We then say a[0] = a[1], and a[1] equals 5, so now a[0] = 5 (yay, we swapped a[j] into a[i], but now we need to swap what was in a[i] back into a[j]). The variable temp still holds the previous i value. So we can just say that a[j] = temp because we are swapping the values, after all. So now your a[i] = a[j] and your a[j] = a[i]. Now go get to sorting!
